# INSTALLATION:

### 1. Unpack the archive

### 2. Move the mod from the archive to the game folder along the path: “\AtomicHeart\Content\Paks\~Mods” (if there is no such folder, you need to create it)

### 3. Run Inject

### 4. Start the game
